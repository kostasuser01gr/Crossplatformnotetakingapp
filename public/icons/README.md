# PWA Icons

This directory should contain the PWA icons for the application.

Required icons:
- icon-192.png (192x192)
- icon-512.png (512x512)

To generate icons, you can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

For now, placeholder icons should be added here before deployment.
